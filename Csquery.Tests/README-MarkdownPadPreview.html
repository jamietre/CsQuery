CsQuery.Test

6/25/2012

The CsQuery test suite can be run in both Visual Studio and NUnit test runners. If you add new tests please follow this convention so it will work in either place.

  1. Every test should inherit from CsQueryTest

  2. A test class should be marked with attributes for both Visual Studio and NUnit.

    [TestFixture, TestClass] public class MyNewTestClass: CsQueryTest {

    }
